# 🛵 Food Delivery Website

A full-stack Food Delivery application built with MERN Stack.

## 🚀 Features
- User Authentication (JWT + bcrypt)
- Restaurant & Menu Listing
- Cart & Order Management
- Real-time Order Tracking (Socket.IO)
- Payment Integration (Razorpay)
- Image Upload (Multer + Cloudinary)
- Admin Dashboard

## 🛠️ Tech Stack
**Frontend:** React.js, Redux, Tailwind CSS

**Backend:** Node.js, Express.js, MongoDB

**Real-time:** Socket.IO

**Other:** JWT, bcrypt, Cloudinary, Multer, Razorpay

## ⚙️ Installation

```bash
# Clone the repo
git clone https://github.com/poonammgambhire/Food_Delivery_Website.git

# Install dependencies
npm install

# Run
npm start
```
